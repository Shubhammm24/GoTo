## 💡 Key Implementation Patterns

### Authentication
```javascript
// Login stores JWT token
const { login } = useAuthStore();
await login(email, password); // Token auto-stored

// All API calls include token
axios.defaults.headers.Authorization = `Bearer ${token}`;

// Protected routes check token & role
<ProtectedRoute requiredRole="customer">
  <BookingPage />
</ProtectedRoute>
```

### Real-time Fare Calculation
```javascript
// Formula: (base + distance×rate + duration×rate) × surge
const fare = (baseFare + distance*perKm + duration*perMin) * surgeMultiplier;
```

### State Management
```javascript
// Simple, scalable Zustand stores
const useAuthStore = create(state => ({
  token: null,
  login: async (email, password) => { ... },
}));
```

### API Integration
```javascript
// Axios with JWT interceptors
const api = axios.create({
  baseURL: 'http://localhost:5000/api',
  headers: {
    Authorization: `Bearer ${token}`,
  },
});
```
