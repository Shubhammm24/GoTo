## 💡 QUICK TIPS

### Development
- Backend runs on `http://localhost:5000`
- Frontend runs on `http://localhost:5173`
- API URL is configured in `.env`
- Hot reload enabled for both
- Real-time changes reflected instantly

### Testing
- Use provided test accounts
- Or register new ones in the UI
- Test all 3 roles
- Check responsive design on mobile
- Verify fare calculations

### Customization
- Modify colors in `tailwind.config.js`
- Change pricing in `BookingPage.jsx`
- Update text in component files
- Add your logo/branding
- Customize email templates

### Deployment
- Build frontend: `npm run build`
- Deploy `backend/` folder
- Deploy `frontend/dist/` folder
- Set environment variables
- Configure database (MongoDB Atlas)
- Set up payment gateways
